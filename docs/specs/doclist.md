# spec: doclist

## Назначение
Браузер конспектов: список всех `.md` в папке направления, тап открывает на чтение (как `doc`).
Напр. iOS Notes.

## Данные
Папка с `.md`-файлами. Можно исключать файлы через `exclude` в артефакте config.md.

## Поведение (UI)
- Список файлов (имя без `.md`), счётчик.
- Тап → чтение файла (рендер `marked`, как `doc`), кнопка назад.

## Правила и крайние случаи
- Папки нет (404) → сообщение.
- Только чтение; файлы добавляются через Obsidian.
- (Прим.: для Sport используется отдельный **редактируемый** браузер программ — см. specs/sport.md, не doclist.)

## Код
`js/kinds/doclist.js`: `renderDocList`.
