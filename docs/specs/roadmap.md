# spec: roadmap

> Формат — [guides/iOS.md](../guides/iOS.md).

## Назначение
Интерактивный план обучения: фазы → майлстоуны → сессии. Напр. iOS Senior+ Roadmap.

## Данные
Один `.md`: структура фаз/майлстоунов/сессий + чек-лист `- [x]` (источник истины прогресса) +
счётчики во frontmatter.

## Поведение (UI)
- Сворачиваемое дерево: фазы → майлстоуны → сессии.
- Тап по сессии: чек-бокс (готово/нет), раскрытие описания/тегов.
- Прогресс считается вживую; на запись синхронизируются: чек-бокс, статус в таблице (✅/⬜),
  счётчики frontmatter (sessions_completed/total/percent, next_session, current_phase/milestone, per-phase Done).

## Правила и крайние случаи
- Источник истины — чек-лист; остальное — производное, пересчитывается при записи.
- Запись = перезапись файла (толерантно к остальному содержимому).

## Код
`js/kinds/roadmap.js`: `parseRoadmap`, `setSession`, `syncRoadmapMeta`, `renderRoadmap`.
