# spec: vocabulary

> Формат файла — [guides/English.md](../guides/English.md). Решение по дате — ADR [0007](../decisions/0007-vocabulary-added-column.md).

## Назначение
Словарь слов/фраз с переводом. Направление English (артефакт `Vocabulary.md`).

## Данные
Один Markdown-файл, таблица `Word | Translation | Mastered | Example | Note | Added`.
`Mastered` = yes/no; `Added` = `YYYY-MM-DDTHH:MM:SS` (МСК). Запись = перезапись всего файла.

## Поведение (UI)
- Карточки слов: слово, перевод, пример (строка `>`), заметка, бейдж `[MASTERED]`/`[LEARNING]`.
- Поиск по слову/переводу/заметке/примеру.
- Сортировка: по алфавиту / по дате добавления + переключатель направления `↑/↓`.
- Добавление (форма: слово, перевод, пример, заметка, «уже выучено») и удаление (с подтверждением).
- Счётчик «N to master» (невыученные).

## Правила и крайние случаи
- Дубль слова (без учёта регистра) — запрет с alert.
- Файла нет (404) → пустой словарь, создаётся при первом добавлении.
- `Added` у старых слов пустой → при сортировке по дате (desc) уходят в конец; мигрирует лениво.
- В ячейках `|` экранируется как `\|`.
