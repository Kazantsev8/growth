# spec: notes

## Назначение
Backlog заметок/идей по направлению. Канал «идея → анализ AI → развитие направления».

## Данные
Один `.md`. Дат-блоки `## ГГГГ-ММ-ДД ЧЧ:ММ` + текст, новые сверху.

## Поведение (UI)
- Поле ввода + кнопка «Добавить заметку» (штамп времени).
- Список карточек (дата + текст), удаление с подтверждением.
- Счётчик заметок.

## Правила и крайние случаи
- Пустой ввод не добавляется.
- Файла нет (404) → пустой список.
- Откат при ошибке записи (возврат заметки в список).

## Код
`js/kinds/notes.js`: `parseNotes`, `serializeNotes`, `renderNotes`.
