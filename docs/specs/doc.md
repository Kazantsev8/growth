# spec: doc

## Назначение
Markdown-справочник, рендерится как есть (только чтение). Напр. English: времена/условные/неправильные глаголы.

## Данные
Один `.md`-файл. Рендер тела через `marked`.

## Поведение (UI)
- Заголовки, таблицы, списки, цитаты — как в Markdown.
- Якоря оглавления: заголовкам проставляются `id` по GitHub-слагу; внутренние ссылки `#слаг` скроллят.

## Правила и крайние случаи
- Файла нет (404) → сообщение «файла нет».
- Только чтение; правки — в Obsidian.

## Код
`js/kinds/doc.js`: `renderDoc` (+ `slugify` из `core/md.js`).
