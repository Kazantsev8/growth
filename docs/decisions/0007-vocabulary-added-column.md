# 0007 — Vocabulary: колонка Added для сортировки

- **Статус:** принято
- **Дата:** 2026-06-13

## Контекст
Нужна сортировка словаря по дате добавления (и по алфавиту). В таблице не было поля даты;
порядок строк как proxy ненадёжен (правки в Obsidian). Пользователь: дату хранить у каждого слова,
отображать необязательно.

## Решение
Добавлена колонка `Added` (дата `YYYY-MM-DDTHH:MM:SS`, МСК, без TZ). Штампуется при добавлении слова.
Старые слова без даты мигрируют лениво (пустое поле при ближайшей перезаписи), при сортировке по дате
уходят в конец (desc). UI: переключатель «по алфавиту / по дате добавления» + направление.

## Последствия
- (+) Честная сортировка, расширяемая схема таблицы.
- (−) Лёгкая ленивая миграция (пустые `Added` у старых строк).
- Формат — [guides/English.md](../guides/English.md); поведение — [specs/vocabulary.md](../specs/vocabulary.md).
