# 0005 — Tasks: файл на задачу, active/archived

- **Статус:** принято
- **Дата:** 2026-06-08

## Контекст
Персональный таск-трекер. Нужны «текущие» и «архив», устойчивость к last-write-wins, читаемость в Obsidian.

## Решение
Одна задача = один `.md`-файл. Папки `active/` и `archived/` внутри `Tasks/`. Закрытие = перенос
файла `active→archived` (реализовано как write+delete, не WebDAV MOVE). id = имя файла `Task-{N}`,
N = `max(active+archived)+1` (без отдельного счётчика). Вложения — подпапка на задачу `Documents/Task-N/`.
Типы: simple/checklist/steps. Статусы todo/in_progress/done с историей переходов во frontmatter.

## Последствия
- (+) Дробная запись (правишь одну задачу — один маленький файл), меньше конфликтов.
- (+) Видно в Obsidian; архив листается отдельно.
- (−) Список читает все файлы при открытии (нет индекса) — приемлемо, пагинация ограничивает отрисовку.
- Детали формата — [specs/tasks.md](../specs/tasks.md) и [guides/Tasks.md](../guides/Tasks.md).
